Once the configuration is saved and Cline is restarted, the following tools will be available:

### 1. **tavily-search** - Real-time Web Search

- **Purpose:** Perform live web searches to find current information
- **Example Usage:** "Search for the latest news about AI developments in 2024"
- **Features:**
  - Real-time search results
  - Multiple search query support
  - Source attribution

### 2. **tavily-extract** - Intelligent Data Extraction

- **Purpose:** Extract specific information from web pages
- **Example Usage:** "Extract the main topics from this research paper"
- **Features:**
  - Smart content extraction
  - Structured data parsing
  - Content summarization

### 3. **tavily-map** - Website Structure Mapping

- **Purpose:** Create structured maps of website content
- **Example Usage:** "Map the structure of this website to understand its organization"
- **Features:**
  - Site structure analysis
  - Content hierarchy mapping
  - Navigation flow visualization

### 4. **tavily-crawl** - Web Crawling

- **Purpose:** Systematically explore and index website content
- **Example Usage:** "Crawl this website to gather comprehensive information"
- **Features:**
  - Systematic site exploration
  - Content indexing
  - Link discovery and analysis
